# task_app

## What's this app ?
Vue.jsの練習用に作ったLaravelアプリ。
JWTによるユーザ認証、Vuexでのストア管理、APIでのCRUD機能実装。

## Environments
Laradock

Laravel: 5.7

PHP: 7.3.11
